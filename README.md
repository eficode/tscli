# Timesheets CLI

A simple CLI for Eficode Timesheets

## Prerequisites

Sign in to [Timesheets](https://timesheets.eficode.fi/) on Chrome.

## Running

`npx @eficode/tscli`

### Listing current tasks

`npx @eficode/tscli tasks`

### Adding an hour report

`npx @eficode/tscli create <id> <duration>`
